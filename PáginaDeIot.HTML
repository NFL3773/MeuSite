Atividade do Boné:

Nessa atividade desenvolvemos um boné com um sensor de acessibilidade para cegos.
Quando a pessoa estivesse utilizando o boné e se aproximasse de algum objeto, ele emitiria alertas sonoros.


Atividade do Carrinho Arduino:

Nessa atividade construímos um carrinho usando Arduino para que ele seguisse uma linha preta no chão. Como é mostrado no vídeo.


Atividade do 4º Bimestre (em grupo):

Nessa atividade construímos um sensor de temperatura para que quando atingisse uma certa temperatura um ventilador acionasse.
